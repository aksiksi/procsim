# Project 2: Checkpoint

## Build

Run `make` in root directory.

## Run

Format: `./procsim -f F -r R -j J -k K- l L -i <trace_file>`

Example: `./procsim -f 4 -r 2 -j 3 -k 2 -l 1 -i gcc.100k.trace`

The output file will have the same name but with the extension `.out` and written to the same directory.

For the example above: `gcc.100k.trace.out`.
